# 🚢 Titanic Dataset – EDA & Feature Engineering

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset and converts raw data into a machine learning-ready format through feature engineering and preprocessing.

---

## 🎯 Objective

* Understand factors affecting survival
* Clean and preprocess data
* Perform feature engineering
* Prepare dataset for ML models

---

## 📂 Dataset

* Titanic dataset containing passenger details
* Features include Age, Fare, Gender, Class, Cabin, etc.

---

## 🔍 Key Analysis

* Fare vs Survival
* Family size vs Survival
* Cabin & Deck impact
* Gender & Age group analysis
* Passenger class & Embarked location

---

## ⚙️ Feature Engineering

* Created family-based features
* Extracted deck from cabin
* Converted categorical variables to numerical
* Created age groups
* Engineered favorable family indicators

---

## 🧹 Data Cleaning

* Handled missing values
* Replaced invalid Fare values
* Removed unnecessary columns

---

## 📏 Preprocessing

* Applied encoding techniques
* Standardized numerical features

---

## 📊 Results & Insights

* Higher fare passengers had higher survival rates
* Females and children had better survival chances
* Cabin availability strongly influenced survival
* Feature engineering improved data quality
